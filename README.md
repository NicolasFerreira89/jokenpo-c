# ✊✋✌️ Jokenpô em C

Projeto desenvolvido em linguagem C com o objetivo de praticar lógica de programação, estruturas condicionais e geração de números aleatórios.

## 🚀 Descrição

Este projeto simula o jogo Jokenpô (Pedra, Papel e Tesoura), onde o jogador escolhe uma opção e o computador gera uma escolha aleatória.

O sistema compara as escolhas e determina o vencedor.

---

## 🧠 Funcionalidades

- Escolha do jogador via entrada de dados
- Geração de jogada aleatória do computador
- Exibição das escolhas
- Lógica para determinar vitória, derrota ou empate

---

## 🛠️ Tecnologias utilizadas

- Linguagem C
- Bibliotecas:
  - `stdio.h`
  - `stdlib.h`
  - `time.h`

---

## 📚 Conceitos aplicados

- Estruturas condicionais (`if/else`, `switch`)
- Geração de números aleatórios (`rand`)
- Entrada e saída de dados
- Lógica de decisão
